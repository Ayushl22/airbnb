# Airbnb Clone Web Application

## Overview
This is a full-stack Airbnb-style web application built with **Node.js**, **Express.js**, **MongoDB**, and **EJS**. The platform allows users to **list properties, book stays, and leave reviews**, emulating core Airbnb functionality.  

The app demonstrates **authentication, authorization, MVC architecture, and cloud deployment** using MongoDB Atlas and Render.

---

## Features
- User authentication and session management using **Passport.js**  
- Property listing creation, editing, and deletion  
- Booking functionality for users  
- Reviews and ratings for properties  
- Responsive UI using **EJS templates**  
- Cloud deployment ready with **Render**  

---

## Tech Stack
- **Frontend:** HTML, CSS, EJS templates  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB Atlas  
- **Authentication:** Passport.js (Local Strategy)
